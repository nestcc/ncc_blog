# 什么是linux内核

* 从功能上讲，内核就是一个`软件`和`硬件`之间交互中间层。
* 也可以理解为一个`lib`，内核给我们提供了很多sys_call的接口。
* 或者是一个Manager，管理计算机的资源。



# 内核实现的策略

* 宏内核
* 微内核



# 核心模块

1. 进程调度、切换
2. 内存管理
3. 虚拟内存管理
4. 网络交互
5. 设备驱动程序  
6. 进程通信机制、锁
7.  其他模块  
   1. 页表的管理
   2. 数据同步
   3. 二进制文件
   4. 时序
